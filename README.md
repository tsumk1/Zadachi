# **Zadachi.** **Five starting tasks**

**First task**

You are given two numbers (either entered by the user or constants in the code). Find the sum of all even integers in the range between these given numbers, inclusive

**Second task**

You are given a number (either entered by the user or a constant in the code). You need to check whether it is negative or positive, and output the result of the check to the console

**Third task**

The code has a string of type const char *. You need to output the length of this string to the console

**Fourth task**

You are given a number (either entered by the user or a constant in the code). Convert it to a string (std::string)

**Fifth task**

Find all “lucky” tickets with numbers in the range from 000000 to 999999. A lucky ticket is one in which the sum of the digits of the first half is equal to the sum of the digits of the second half, for example: 526265, 145226
